Gym Supplement Store

A personal e-commerce project built with HTML, CSS, JavaScript, PHP and MySQL.

Features:
-Customer and admin login systems

-Product management (add, edit, delete)

-Shopping cart and order system

-Contact form with messages stored in the database

-Password hashing for security

How to Run:
-Install XAMPP

-Put the project in C:\xampp\htdocs\

-Import the database and set up your .env file

-Start Apache and MySQL, then open http://localhost/gym-supplement-store
